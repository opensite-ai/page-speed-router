## [0.0.1] - 2026-03-10

### Added

- Initial release with performance-focused React routing hooks for custom rendering applications
- Tree-shakable exports with core and hook-level entrypoints plus UMD build for CDN usage.
- 2 hooks included:
  - **Routing**: `useNavigation`
  - **URL Data**: `useUrl`
